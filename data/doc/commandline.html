Command Line

Insensitive offers a Bruker style command line at the bottom of the main window. A subset from the commands known from the XWin-NMR and TopSpin applications are available for users familiar with these programs to quickly access functions for spectrum acquisition, processing, and system variables. The commands are not case-sensitive. A list of commands is given below, which take no arguments:

A number of system variables can be changed by the following commands. Most take one argument, which unless stated is a floating point value. Variables that are proprietary to Insensitive are marked with and asterisk (*).

GB Adjust maximum position for a gaussian window function.

GPT Gradient pulse duration. (*)
LB Adjust width for a gaussian window function.
NS Number of scans (number of phase cycles).
NUC1 Type of I-spin nucleus. Possible values are 1H, 13C, 15N, 19F, 29Si, and 31P.
NUC2 Type of S-spin nucleus. Possible values are 1H, 13C, 15N, 19F, 29Si, and 31P.

PHC0 Zero order phase shift in degrees.

Actions on the spin system can be issued using the following propietary commands in Insensitive or by applying product operators directly, optionally preceded by a numerical factor. The available product operators for pulses are Ix, Iy, Sx, Sy, and derived operators for specific spins in the form of e.g. I1x or S2y. The numerical factor is given in units of 1°. If no coefficient is given, a 90° pulse is assumed. Insensitive also understands "pi" and π = 180°, π/2 = 90°, and π/4 = 45°. For chemical shift the operators Iz and Sz (and spin-specific operators like e.g. I1z) can be used for the given delay time. J-coupling is applied using the operators for longitudinal two-spin order, e.g. 2I1zI2z, 2I2zS3z, 4I1zI2zS4z, or 8I1zI2zS3zS4z. Alternatively the operators IzIz, SzSz, and IzSz perform coupling for I-spins, S-spins, and all spins, respectively.

Command Function

animr Start animation of precession, including relaxation.

prec Free precession performed for delay time d or an evolution time given by an optional numerical parameter.
precr Free precession including relaxation performed for delay time d or an evolution time given by an optional numerical parameter.

pfg A pulsed field gradient is issued.

ssdisp Open individual spin precession window.
