Creating a Spin System

To begin the simulation a spin system has to be created. At first launch of Insensitive this consists of two spins of spin type I with a Larmor frequency of Ω = 0 Hz (in the rotating frame) and no scalar coupling. The default distance between the spins is r = 0.30 nm, which corresponds to a dipolar coupling constant of b = -4450.69 Hz. To create an instructive simulation these parameters need to be adjusted. Spin systems, including the current spin state, can be opened and saved as Insensitive Spin System (*.iss) files using the "File" menu.

Setting the spin number

A maximum of 4 spins of two possible spin types, I and S, can be simulated. The number is limited by computing power of state-of-the-art computers. Click "Add Spin"/"Remove Spin" or edit the text field labelled "Spins" to change the number of spin in the spin system. "Remove Spin" removes the selected spin indicated by a blue glow. The spin system is displayed as red (I spins) and blue (S spins) spheres, which are connected by straight lines that symbolise coupling. The user can select whether the connections display scalar coupling (J), dipolar coupling (b), or distance (r).

Energy Levels

On the right hand side the energy levels of all spins are displayed as black lines, as well as observable single quantum transitions between states as red arrows. The transition arrows do no distiguish between degenerate levels. The levels are labelled with the spin states α or β for each participating spin, or, in the case of strong coupling, as a wave function ψ with a running number. For magnetically equivalent spins the symbols from the singlet-triplet basis are being used.

Setting the spin type

The spin type can be changed between I and S by double tapping the respective spin icon. The spin types for I and S can be assigned to a number of common nuclei: 1H, 13C, 15N, 19F, 29Si, and 31P. The standard setting is 1H and 13C for I and S spins, respectively. This setting will affect the gyromagnetic ratio of the spins, but natural abundance will be ignored. Note that 15N and 29Si have a negative gyromagnetic ratio.

Setting the Larmor frequency

A spin is selected by single clicking the spin icon. To change the Larmor frequency (and thus the chemical shift) of the active spin enter the new value in the text field indicated by Ω and press enter. The units can be displayed in Hertz or degrees per second (which can be set with the popup button "Units").

Setting the coupling constants

To set up coupling constants or spin-spin distances first enter the value for the constant that is currently being displayed (J, b or r). Then click one spin and drag a connector to the coupling partner to set the value. Alternatively, if only two spins are present the value can be set by pressing enter. If the distance is set the dipolar coupling constant will be calculated automatically and vice versa.

Resetting and rotating the spin system

Clicking "Reset Spins" will reset all spin types to I and set the chemical shifts and coupling constants to 0 Hz and the distance between spins to 0.30 nm. "Rotate" will change the indices of the spins so that spin 1 becomes spin 2 and so on, with the highest index being set to 1.

Considerations

Avoid setting the frequencies to integer values, as the spin system will appear unchanged after the default delay time τ = 1.0 s. For an instructional simulation it is advisable to set the frequencies to values that are smaller than typical chemical shifts and coupling constants (e.g. Ω1 = 0.2 Hz, Ω2 = -0.7 Hz, J = 0.25 Hz). This way the free evolution is easy to observe in steps and the effects of strong coupling are most pronounced. A good starting value for the spin-spin distance is 0.3 nm between two I spins, 0.2 nm between I and S spins and 0.13 nm between two S spins. For acquiring the spectrum set the dwell time to 0.1 s and the data poits to 512 (with zero filling) or 1024 (without zero filling) for good spectral resolution.
