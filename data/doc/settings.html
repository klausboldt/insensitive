Settings

The Settings panel offers to change all parameters that do not directly concern the spin system, which is set up in the Spin Network Editor.

Pulse settings

In the pulse settings the flip angle and phase of the rf-pulses can be set, either by entering the angles as a numerical value between 0 and 360° or by setting the slider in intervals of 15°. Alternatively to the flip angle pulse duration, strength, and frequency can be set to manipulate the pulse bandwidth. Presets suitable for spins that are resonant between -10 and +10 Hz can be selected by clicking "Hard", "Soft", or "Selective". To create shaped pulses a number of standard pulse envelopes can be selected from the Shape menu. The pulse will only be applied to the spins which are set to be resonant at the bottom of the panel, which overrules the resonance conditions set above. Resonance can be set for single spins as well as for groups of spins of the same spin type (I or S).

The "Pulse Shape" module, which can be opened from the main window's toolbar, displays the pulse envelope and corresponding excitation profile of the selected pulse shape. The left panel shows the pulse shape and allows to set the pulse duration Δt, either numerically or by changing the slider. The right panel displays the pulse's power spectrum and allows to set the pulse's resonance frequency Ω at which the flip angle is equal to the one set in the settings. If off-resonance effects are disabled in the preferences the Fourier transform of the pulse shape is used as an approximation to calculate the flip angle.

Free evolution settings

In this panel the spins' behaviour during free evolution can be set. "Strong coupling" activates strong coupling between spins of the same type. "Dipolar relaxation" activates relaxation via dipole-dipole couplings in a system of two or more spins. By checking "Animation" chemical shifts and scalar couplings will be continuously applied to the spin system and is displayed as an animation. If "Relaxation with evolution" is checked, relaxation will be taken into account for free evolution, both when clicking the "Free Evolution" button and during animation. The delay time is the step width for which precession, coupling and relaxation is applied when the respective buttons are pressed.

Relaxation is simulated either empirically by applying the relaxation constants T1 and T2 ("Dipolar relaxation" off) or by calculating the relaxation times from the dipolar coupling constants and the correlation time τc ("Dipolar relaxation" on). The latter is necessary to simulate the nuclear Overhauser effect. T2 must be smaller or equal to T1. Depending on the settings either the relaxation times or τc is not used for the simulation.
By activating "Random jitter" Insensitive simulates dephasing through random fluctuations of the Lamor frequency, which can be caused by diffusion and inhomogenieties of the magnetic field. The coherence cannot be refocussed by spin echo sandwiches, because the simulation is limited by the density matrix approach.

There are two possible ways to achieve decoupling in Insensitive: If "Decoupling I-Spins/S-Spin" is checked, 180° pulses with a 90° phase shift to the phase set in the pulse settings will be applied during coupling and free evolution. This constantly refocusses the selected spin type and reverses the effects of J-coupling. Alternatively, a spin lock field can be applied, which works as a continuous rf-field on both spin types. In this case the precession will happen around the axis perpendicular to the set pulse phase. In order to observe the ROE with T1ρ relaxation or cross polarisation "dipolar relaxation" has to be checked as well.

Gradient settings

Pulsed field gradients along the z-axis will be applied with a strength and duration set in this panel. The values for gradient strength may vary between +32000 and -32000. When enabling "Diffusion" the loss of magnetisation from dephasing due to diffuion in the gradient field is accounted for.

Acquisition settings

Insensitive is able to acquire one-dimensional spectra. The FID will be recorded for the 2n data points spaced by the dwell time, which determines the spectral width and resolution of the spectrum. The "Noise level" setting simulates random noise, which is added to the spectrum. The number entered in the percentage of the noise compared to the signal of one uncoupled spin of type I.

If "Zero filling" is activated a row of zeros will be appended to the acquired FID, effectively doubling the spectral resolution. Changing "Zero filling" or the number of data points will reset the spectrum window.

Acquisition of the spectrum is started by pressing the "Acquire" button on the Spin State panel. If "after next pulse" is checked the acquisition will be started after the next rf-pulse.

Display setting

From the "Vectors" popup button it can be chosen to display all observable coherences each as a single vector, the magnetic moments of each spin or the total angular momentum that is recorded as the free induction decay as one single vector for each spin type. When selecting "moments" pure antiphase magnetisation will collapse to zero.
The product operators can be set to be displayed either in cartesian or spherical operator basis.
The observable "simple" single-quantum coherences are marked in the density matrix if "Colour 1st order coherences" is checked. (-1)-coherences are marked in green while (+1)-coherences are marked in red.
"Font size" allows to display the density matrix in different sizes.
