User's Manual

Insensitive (Incredible Nuclear Spin EvolutioN SImulation Tool Intended for Visual Education) is a simulation of nuclear magnetic resonance spectroscopy on the level of a spin ensemble density matrix. Every step in the simulation can be controlled, a fact that is highly instructive, but can on occasion require less efficient algorithms than simulations aimed at realistic spectra of complete molecules. A complete simulation usually consists of four distinct steps:

  1. Set up a spin system
  2. Manipulate the spin state
  3. Record and run a pulse sequence
  4. Acquire and process a spectrum

Each of these steps is handled in a dedicated panel in the main window of the application. Each panel can be accessed via a button in the toolbar of the main window.

The toolbar consists of 8 buttons. The first four (Spin System, Spin State, Pulse Sequence, Spectrum) access the main steps of the simulation. Four more buttons access additional windows that control advanced functions in Insensitive: the Matrix Composer to directly manipulate the density matrix, the Pulse Shape module to view and edit the pulse shape and excitation profiles, the Single Spins module to help visualise the translation between the behaviour of single spins and the ensemble magnetisation described by the density matrix, and finally this Tutorial help file. The Presentation Editor that allows to build simple full screen presentations including all available models and animations is currently not implemented.

Command Line Shortcuts

In addition to the graphical user interface Insensitive offers a command line at the bottom of the main window. It allows to apply product operators directly to the spin state. In addition, Insensitive unerstands many commands from Bruker NMR applications (XWin-NMR, TopSpin) and a number of proprietary commands to access basic functions of the application.
