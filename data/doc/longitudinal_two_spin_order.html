Longitudinal two-spin order

The longitudinal two-spin order is a state of two linked populations 2I1zI2z. If spin 1 is in α-state (red) the second spin is also in that state (blue). The same holds true for the β-state.

Since this operator only represents coupling information and no excess population the total magnetisation is zero.

Longitudinal two-spin order is found as an intermediate state in coherence transfer. It's exponential operator exp(0.5 J τ 2I1zI2z) is the operator for weak, scalar coupling.
