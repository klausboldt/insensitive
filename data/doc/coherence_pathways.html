Coherence Pathways

A radiofrequency pulse changes coherences in a spin system from one coherence order to another. After the first pulse, which creates only singe quantum coherences from the state of thermal equilibrium, every pulse is in principle capable of transforming every coherence into every other coherence possible in that spin system. The most convenient way to describe this is by using spherical tensor operators, which directly display the coherence order.

A coherence pathway is the sequence of coherence orders that the magnetisation evolves through during a pulse sequence. All experiments start with zero order coherence (z-magnetisation) and should end with a coherence order of -1, which is by convention the one that is detected by the quadrature detector. Without quadrature detection the +1 coherences would be equally detectable, all higher orders are not correlated with observable magnetisation.

For an NMR experiment it is not desirable to allow every possible coherence pathway to evolve. If one considers the NOESY and DQF-COSY experiments, both have essentially the same pulse sequence of three pulses divided by delay times of varying length.

The outcome of the experiment is decided by a correct selection of coherence pathways, usually done by phase cycling, pulsed field gradients, or a combination of both.

The coherence pathway is displayed like a musical score below the pulse sequence with the coherences being drawn as black lines that switch the order with each pulse. The diagrams are usually idealised and show only the relevant pathways for the experiment. Depending on the spin system, delay times and settings the results from the computations can vary.

For a spin system with n spins there are 2n+1 possible coherences orders, negative and positive orders and the zero order coherence. For a pulse sequence of p pulses the maximal number of possible coherence pathways is (2n+1)p. This number of calculations needs to be performed for every phase cycle in order to get a complete description of the coherence pathways. While the number can be reduced by constraints like the original coherence order being zero, the high number of computations limits the size of pulse sequences for which a coherence pathway can be displayed in Insensitive.
