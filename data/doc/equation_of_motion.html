Equation of motion

If a wave function is an eigenfunction of the Hamiltonian it does not change over time, except for a complex phase factor. If this is not the case as with a superposition state, where the coefficients of the linear combination change as well. The equation of motion or time-dependent Schrödinger equation can be used to calculate its evolution in time.

If this first order differential equation is solved one gets an exponential factor that expresses the temporal evolution of a wave function.

This exponential operator looks very similar to the rotational operator given below. The Hamiltonian is substituted by a cartesian component of a spin angular momentum operator while the time t is substituted by an angle β.

In any rotational motion the time of free evolution can be easily converted to an angle be applying the formula ω0t = β with ω0 being the spin's Larmor frequency. The Hamiltonian of a simple spin-1/2 can be written as the spin angular momentum operator multiplied by ω0. Thus, a temporal evolution of a nuclear spin that is not in an eigenstate of the current spin Hamiltonian can be described as a rotation around a fixed axis. This can be both free precession as well as rf-pulses.

The exponential of an operator's matrix representation can be easily calculated if the matrix is in its base-diagonal form, where the diagonal matrix elements are the operator's eigenvalues and all off-diagonal matrix elements are zero. Then the matrix exponential is simply calculated by taking the exponential of the eigenvalues.
