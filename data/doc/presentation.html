Presentation

Insensitive has the possibility to create simple presentations for lectures and talks. All simulations and representations that are part of Insensitive can be freely positioned on a series of pages with an aspect ratio of 4:3 and presented in full screen.

On the right hand side of the presentation window there are three panels that can be selected from the buttons at the top, Objects, Images, and Settings. The objects panel shows a list of all available Insensitive objects that can be used in the presentation, which are:

  • Spin network
  • Vector model (I spins)
  • Vector model (S spins)
  • Energy level diagram
  • Density matrix
  • Product operators
  • Pulse sequence
  • Coherence pathways
  • Phase cycling table
  • Spectrum
  • Pulse shape
  • Pulse power spectrum
  • An image
  • A text field
  • Control buttons

The image panel provides access to all images that are included with the Insensitive binary as part of the tutorial. Objects from both these panels can be placed onto the canvas by drag and drop.
The settings panel offers the same settings as the main window for the currently selected object. Depending on the object some setting may have no effect.

Working with frames

If an Insensitive object is placed on the canvas and selected by clicking, a black frame appears around the object. If the objects retain their aspect ratio, for example images, the frame can be larger than the visible object, which appears centered. The frame can be resized by clicking and dragging one of the corners or edges. Frames can be movd by clicking on a non-responsive part of a frame and dragging. The current position relative to the top left corner is displayed during moving.
Frames can be deleted by clicking "Delete Frame" in the toolbar or pressing the backspace key. They can be reordered by selecting the "Bring to Front" or "Send to Back" buttons. When clicking an a stack of objects only the uppermost frame will be selected.
The font size and type can be changed for text fields or product operators with the "Fonts" panel from the toolbar.

Loading external data

External data can be loaded into an object by selecting it and clicking "Import Content" in the toolbar. Depending on the type of object Insensitive will load different file types. Possible files are images, spin system files (*.iss), pulse program files (*.ipp) and spectrum files (*.igg). Spin systems, pulse programs and spectra cannot be created or edited inside the presentation window, but must be saved from the respective windows in Insensitive and then imported.

Linking frames

Each frame contains a complete Insensitive simulation engine that runs independently from the others. Frames can be linked by dragging the mouse from the grey circle with the chain symbol in the upper right corner of an activated frame and dropping it on another that shares the same data type (see image above). The settings for the frame from which the connection is drawn will be deleted and the frame will share the simulation with the target frame. If part of another shared simulation the frame will leave its previous network.
Examples for connections are a vector model, density matrix, and product operators sharing the same spin system, a vector model sharing the engine with the pulse shaper frames for selective pulses, or pulse sequence, coherence pathway and phase cycling table showing the same pulse program.
The controls, which initially display buttons to manipulate the spin system, change behaviour if they are connected to either spectra or the pulse shaper objects.
Press and hold the CTRL button to see which frames are linked.

Adding and removing slides

Slides can be added or deleted by clicking the toolbar buttons "Add Page" or "Remove Page". A new page is inserted after the currently displayed slide. A slide can be selected from the "Select Page" dropdown menu.

Full screen mode

Clicking on "Play" in the toolbar starts the presentation in full screen mode. Editing the slides is not possible in full screen. The presentation starts with the currently active slide. The previous or next slide can be selected by pressing the left or right cursor key. Full screen mode can be closed by clicking the button in the bottom left corner or pressing the escape key.

Opening and saving presentations

Presentations can be saved in the *.spinlecture format from the toolbar. A copy of Insensitive 0.9.13 or higher is necessary to open the files. All data is contained inside the presentation file and imported content does not need to be kept separately.
Currently there is no automatic saving and the files need to be overwritten when a new version is saved.
