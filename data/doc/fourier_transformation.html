Fourier transformation

The Fourier transformation is a mathematical tool to transform a function in time-domain (the FID) into a function in the frequency domain (called a spectrum). Mathematically it expresses any function of time f(t) as an infinite series of complex sine and cosine functions. The amplitude function F(ω) is the corresponding spectrum.

The trigonometric functions are conveniently written in the form of Euler's identity

Technically the Fourier transformation has to be performed after converting the analogue signal from the spectrometer into a digital FID. The resolution is determined by two quantities, the number of data points and the sampling rate or dwell time τsample, which is defined as the distance between two data points. Increasing the number of data points causes a higher resolution, since a minimum of three data points is necessary to resolve a peak. Shorter dwell times imply a wider spectral range, more data points per unit time and therefore a higher resolution. Frequencies larger than 1/2τsample cannot be resolved by digital Fourier transformation.

It is most useful to memorise certain Fourier pairs for using NMR:

A single sine or cosine function transforms into a δ-function at the oscillation's frequency.

The exponential decay exp(-wt) transforms into the Lorentz function with the full width at half maximum determined by 1/w. This causes any dampened oscillation to transform into a Lorentzian signal at the oscillation's frequency following the convolution theorem.

A rectangular function transforms into the sinc-function sin(x)/x. This is put to use when applying rf-pulses to excite all spins at the same time, but also leads to artefacts in the spectrum when recoding the FID is stopped before the system is fully relaxed.

A Gaussian function in the time domain transforms into a Gaussian function in the frequency domain. This is useful for selective pulses that only excite one spin in the ensemble.

A periodic δ-function, called a comb function, with the period T transforms into a comb function with the period 1/T. This is the cause of high frequency peaks appearing at the low frequency end of the spectrum and vice versa if the sampling rate is too low.

For 2D spectra the Fourier transformation must be performed on an experiment with two variable time domains.
