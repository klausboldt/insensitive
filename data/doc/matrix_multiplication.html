Matrix multiplication

There is more than one way to multiply two matrices. The term matrix product usually refers to the product where the sum of each element i of the m-th row of the first matrix multiplied with element i of the n-th column of the second matrix forms the element m,n of the matrix product. This is only defined if the number rows in the first matrix equals the number of columns in the second matrix. Matrix multiplication is used in the matrix formulation of eigenvalue equations in quantum mechanics.

The direct product or Kronecker product of two arbitrary matrices m x n and p x q results in an expanded mp x nq matrix. It is used to create the product operators in NMR.

Matrix multiplication is generally not commutative.
